# AI Centre of Excellence (AICOE) Starter Pack

A practical set of resources to launch and support an internal AI Centre of Excellence.

## Contents

| # | Document | Purpose |
|---|----------|---------|
| 1 | [AICOE Charter](./01-AICOE-Charter.md) | Purpose, scope, principles, governance, and operating model |
| 2 | [Resource Hub Index](./02-Resource-Hub-Index.md) | Categorized links to public and internal resources |
| 3 | [Governance Checklist](./03-Governance-Checklist.md) | Responsible AI, privacy, security, approvals, and review |
| 4 | [Copilot Adoption Playbook](./04-Copilot-Adoption-Playbook.md) | Internal rollout guide for GitHub Copilot |
| 5 | [Leadership Deck Outline](./05-Leadership-Deck-Outline.md) | Slide-by-slide outline for executive presentation |

## How to Use

1. **Starting an AICOE?** Read the Charter first, then the Resource Hub.
2. **Rolling out Copilot?** Follow the Adoption Playbook.
3. **Need governance sign-off?** Use the Governance Checklist.
4. **Presenting to leadership?** Build your deck from the Outline.

## Contributing

Update these documents as the AICOE evolves. All materials are markdown for easy collaboration and version control.
