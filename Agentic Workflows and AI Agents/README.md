## How to Use

1. **New to agentic AI on GitHub?** Start with [01-Copilot-Coding-Agent-Overview](./01-Copilot-Coding-Agent-Overview.md) to understand the model before rolling anything out.
2. **Ready to pilot?** Use [02-Configuring-Agentic-Workflows](./02-Configuring-Agentic-Workflows.md) to set up your first well-scoped agent tasks.
3. **Extending agent capabilities?** Reference [03-MCP-Overview-and-Registry-Setup](./03-MCP-Overview-and-Registry-Setup.md) before connecting agents to internal tools and data.
4. **Scaling beyond a pilot?** Apply [04-Agent-Guardrails-and-Governance](./04-Agent-Guardrails-and-Governance.md) so autonomy grows alongside proportional oversight.
5. **Looking for concrete wins?** Explore [05-Use-Cases-and-Custom-Agents](./05-Use-Cases-and-Custom-Agents.md) for patterns teams have found high-value.
6. **Proving it out to leadership?** Use [06-Security-and-Effectiveness-Measurement](./06-Security-and-Effectiveness-Measurement.md) to build a security review and a metrics story together.

## Suggested Rollout Sequence

- **Week 1-2:** Educate stakeholders, pick 2-3 pilot repositories, and define success criteria.
- **Week 3-4:** Configure agent environments, custom instructions, and a small MCP tool allowlist.
- **Week 5-8:** Run pilot use cases (issue triage, PR generation) with human-in-the-loop review on every agent PR.
- **Week 8+:** Formalize guardrails, expand to more teams, and publish an effectiveness scorecard.
