# Plans and Licensing Guide

## Overview

GitHub offers multiple plans to fit teams of all sizes — from individual developers to large enterprises. This guide breaks down each plan so your OpCo can choose the right fit.

---

## Plan Comparison

| Plan | Price | Target Audience | Key Features |
|------|-------|-----------------|--------------|
| **Free** | $0/user/mo | Individual developers, open source | Unlimited public/private repos, 2,000 Actions minutes/mo, community support |
| **Team** | $4/user/mo | Small-to-mid teams | Everything in Free + protected branches, code owners, 3,000 Actions minutes/mo |
| **Enterprise Cloud (GHEC)** | $21/user/mo | Large organizations | Everything in Team + SAML SSO, SCIM, audit log streaming, advanced compliance, data residency options |
| **Enterprise Server (GHES)** | $21/user/mo | Regulated/on-prem orgs | Same feature set as GHEC, self-hosted on your infrastructure |

---

## Licensing Model

- **Per-user licensing** – Each user who accesses GitHub needs a license (seat).
- **Active committer billing** – Advanced Security is billed per unique committer, not per seat.
- **Annual contracts** – Enterprise customers typically purchase annual agreements with volume discounts (10–20% off list).
- **True-up model** – Seats can be added mid-term; you're billed for additional users at the next renewal or true-up period.

---

## Choosing the Right Plan

| If your OpCo needs… | Recommended Plan |
|---------------------|-----------------|
| Basic collaboration for a small team | **Team** ($4/user/mo) |
| SSO, compliance, audit logging | **Enterprise Cloud** ($21/user/mo) |
| On-premises or air-gapped deployment | **Enterprise Server** ($21/user/mo) |
| AI-assisted development | Add **Copilot Business** or **Enterprise** |
| Application security scanning | Add **Advanced Security** |

---

## Negotiation Tips

- Volume discounts are available for 500+ seats.
- Bundle Copilot + Advanced Security for better per-seat pricing.
- Multi-year commitments can unlock additional savings.
- Always work with your GitHub account representative for custom quotes.
