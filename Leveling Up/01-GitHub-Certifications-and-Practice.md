# GitHub Certifications and Practice

## Overview

GitHub offers a growing portfolio of professional certifications that validate skills across the platform. Whether you're a developer, admin, or AI practitioner, there's a certification path for you.

---

## Available Certifications

| Certification | Focus Area | Level |
|---------------|-----------|-------|
| GitHub Foundations | Core GitHub concepts, collaboration, and workflows | Beginner |
| GitHub Actions | CI/CD pipelines, workflow automation | Intermediate |
| GitHub Advanced Security | Code scanning, secret scanning, supply chain security | Intermediate |
| GitHub Administration | Enterprise management, policies, identity | Intermediate |
| GitHub Copilot | AI-assisted development, prompt engineering | Intermediate |
| **GitHub Certified Agentic AI Developer** | Agentic AI patterns, MCP, Copilot extensions | Advanced |

## Free Practice Tests — ghcertified.com

[ghcertified.com](http://ghcertified.com/) provides free, community-driven practice exams for all GitHub certifications.

### What's Included

- Practice questions aligned to official exam objectives
- Timed test simulations
- Detailed answer explanations
- Progress tracking across attempts

### How to Use

1. Visit [ghcertified.com](http://ghcertified.com/)
2. Select the certification you're preparing for
3. Take a practice test under exam conditions
4. Review explanations for any missed questions
5. Repeat until consistently scoring above 80%

## Preparation Tips

1. **Use official learning paths** on [learn.github.com](https://learn.github.com/) alongside practice tests
2. **Hands-on practice** is essential — create repos and experiment with features
3. **Join study groups** in your organization or the GitHub Community
4. **Review the exam guide** for each certification to understand the domain breakdown
5. **Take multiple practice tests** to identify knowledge gaps
