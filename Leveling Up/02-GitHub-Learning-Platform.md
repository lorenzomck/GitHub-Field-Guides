# GitHub Learning Platform

## Overview

GitHub and Microsoft provide comprehensive, structured learning paths for developers at every level. From beginner fundamentals to advanced agentic AI development, these platforms offer self-paced training aligned with certification objectives.

---

## learn.github.com

**Link:** [https://learn.github.com](https://learn.github.com/)

The official GitHub learning hub consolidates all training resources in one place.

### What You'll Find

- **Certification learning paths** — Structured modules mapped to each GitHub certification
- **Role-based tracks** — Paths for developers, admins, security engineers, and leaders
- **Hands-on labs** — Browser-based exercises using real GitHub repositories
- **Progress tracking** — Track completion across modules and paths

### Recommended Learning Paths

| Path | Duration | Outcome |
|------|----------|---------|
| GitHub Foundations | ~8 hours | Ready for Foundations certification |
| GitHub Actions | ~10 hours | Ready for Actions certification |
| GitHub Advanced Security | ~8 hours | Ready for GHAS certification |
| GitHub Administration | ~10 hours | Ready for Admin certification |
| GitHub Copilot | ~6 hours | Ready for Copilot certification |

## Building a Team Learning Plan

### For Individuals

1. Assess current skill level using [ghcertified.com](http://ghcertified.com/) practice tests
2. Choose a certification target
3. Complete the corresponding learning path on [learn.github.com](https://learn.github.com/)
4. Supplement with hands-on practice in real repositories
5. Schedule and take the certification exam

### For Teams

1. Identify skill gaps across the team
2. Assign certification tracks based on roles (dev, admin, security)
3. Set up a shared study cadence (e.g., 2 hours/week)
4. Use GitHub Discussions or Teams channels for peer support
5. Celebrate completions and track progress centrally
