# Courses and Video Training

## Overview

Structured video courses complement hands-on practice by providing expert-led walkthroughs of concepts, workflows, and best practices. This document highlights key courses for GitHub Copilot and agentic AI development.

---

## Practical GitHub Copilot (LinkedIn Learning)

**Link:** [Practical GitHub Copilot — Write, Modify, and Debug Code](https://www.linkedin.com/learning/practical-github-copilot-write-modify-and-debug-code)

### Course Overview

This course teaches developers how to use GitHub Copilot effectively within VS Code for everyday development tasks.

### Topics Covered

- Setting up GitHub Copilot in VS Code
- Writing code with inline suggestions
- Using Copilot Chat for explanations and refactoring
- Debugging with AI assistance
- Modifying and improving existing code
- Best practices for prompt engineering in the IDE

### Who Should Take This

- Developers new to GitHub Copilot
- Teams rolling out Copilot and needing structured onboarding
- Anyone looking to improve their Copilot workflow

### How to Access

- Available with LinkedIn Learning subscription
- Many organizations provide LinkedIn Learning access — check with your L&D team

## VS Code Agentic Development Courses

**Link:** [VS Code announcement on X](https://x.com/code/status/2070145196545753145)

### What's New

VS Code has released courses focused on agentic development patterns — building AI-powered workflows that go beyond simple code completion.

### Topics Include

- Agent mode in VS Code
- Multi-step task automation with Copilot
- Building and using MCP servers
- Custom instructions and workspace context
- Agentic coding workflows and patterns

### Why It Matters

Agentic development represents the next evolution of AI-assisted coding. Instead of single-turn suggestions, agents can:

- Plan and execute multi-file changes
- Run terminal commands and interpret results
- Iterate based on test failures and linting errors
- Integrate with external tools via MCP

## Recommended Learning Sequence

For teams adopting Copilot and agentic workflows, we recommend this progression:

```
1. Foundations (Week 1-2)
   └── Practical GitHub Copilot (LinkedIn Learning)
       └── Copilot basics, inline suggestions, chat

2. Intermediate (Week 3-4)
   └── VS Code Agentic Development courses
       └── Agent mode, multi-step tasks, MCP

3. Advanced (Week 5+)
   └── Agentic AI Developer certification prep
       └── Building extensions, responsible AI, production patterns
```

## Running a Team Watch Party

Video courses are more effective when watched together:

1. **Schedule weekly sessions** — 30-60 minutes, watch one module together
2. **Discuss and practice** — Pause to try concepts in a shared repo
3. **Assign follow-up exercises** — Use GitHub Skills or custom exercises for reinforcement
4. **Track progress** — Note which modules are complete and gather feedback
5. **Share discoveries** — Create a team channel for tips and tricks found during practice
