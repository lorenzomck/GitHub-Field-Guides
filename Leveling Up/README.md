## Introduction

Whether your team is pursuing GitHub certifications, building internal training programs, or exploring the latest in agentic AI development, this pack brings together the best resources in one place. Use it to plan learning paths, run enablement programs, and keep skills current as the platform evolves.

## How to Use This Pack

1. **Getting certified?** Start with document 01 for certification paths and free practice tests.
2. **Building a learning program?** Documents 02 and 03 cover official platforms and interactive tools.
3. **Want video training?** Document 04 has curated courses on Copilot and agentic development.
4. **Transitioning from Classroom?** Document 05 covers the sunset timeline and alternatives.
