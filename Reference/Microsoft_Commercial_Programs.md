# Microsoft Commercial Programs Reference

**Purpose:** Vital intelligence for account planning and commercial discussions with Microsoft customers

---

## MACC (Microsoft Azure Consumption Commitment)

### Definition
A **MACC** is a contractual commitment to consume a certain dollar amount of Azure services over a defined period (typically 1-3 years). It's a spend commitment, not a prepayment.

### Key Facts
- MACC is a prerequisite for ACD (Azure Committed Discount)
- **Without a MACC, there is no ACD**
- MACC size affects the level of ACD a customer receives
- MACC consumption includes GitHub metered products (Copilot, GHAS metered, Actions)

### Why It Matters for GitHub
- GitHub consumption (metered products) counts toward MACC fulfillment
- Customers with large unused MACC may prefer metered GitHub products
- MACC helps justify GitHub investment as "already committed Azure spend"

## ACD (Azure Committed Discount)

### Definition
The **Azure Committed Discount (ACD)** is a percentage discount applied to Azure consumption for customers with a MACC. The discount percentage increases with MACC size and commitment level.

### Key Facts
- ACD is dependent on having a MACC
- **No MACC = No ACD**
- ACD applies to metered billing only
- Typical ACD percentages: 10-30%+ depending on MACC size

### What ACD Applies To

| Product | Pricing | ACD Applied? |
|---------|---------|--------------|
| **Copilot Business** (metered) | $19/user/month | ✅ YES - ACD reduces monthly cost |
| **GHAS Metered** (Code Security) | ~$30/user/month | ✅ YES - ACD reduces monthly cost |
| **Actions** (metered) | Per-minute | ✅ YES - ACD reduces usage cost |
| **GHE License** (EA/committed) | Per-seat/year | ❌ NO - EA pricing, not metered |
| **GHAS Licensed** (EA/committed) | Per-seat/year | ❌ NO - EA pricing, not metered |

### Commercial Relevance

**When discussing metered pricing:**
> "Your ACD would apply to this, so if you have a 28% ACD, your effective price for Copilot Business would be ~$13.68/user/month instead of $19."

**Discovery question:**
> "Do you have a MACC with Microsoft? If so, what's your ACD percentage?"

## Unified Support

### Definition
**Microsoft Unified Support** is Microsoft's enterprise support program that provides comprehensive support across all Microsoft products and services, including access to dedicated resources and proactive services.

### Key Indicators

| If customer has... | Then they have... |
|--------------------|-------------------|
| **Unified Support** | A CSAM (Customer Success Account Manager) |
| **A CSAM** | Unified Support |
| **Unified Support Credits** | Budget for services engagements |

### Unified Support Credits

Customers with Unified Support receive **credits** that can be used for:
- Expert Services engagements
- Partner augmentation
- FastTrack programs  
- Proactive support hours

**GitHub Relevance:** We can request Unified Support credits be allocated to GitHub Expert Services or Partner Services (e.g., BoxBoat, CEI).

### Discovery Questions

> "Does your organization have Microsoft Unified Support?"
> "Who is your CSAM?" (If they have one, they have Unified Support)
> "Do you have Unified Support credits available that could be applied to GitHub services?"

## Commercial Strategy Implications

### For Metered Pricing Recommendations

1. **Ask about MACC first** - determines if ACD applies
2. **Get ACD percentage** - affects true cost of metered products
3. **Calculate effective price** - List price × (1 - ACD%)
4. **Position against MACC fulfillment** - "This consumption counts toward your commitment"

### For Expert Services Funding

1. **Ask about Unified Support** - source of services credits
2. **Identify CSAM** - internal advocate for credit allocation
3. **Quantify available credits** - "Is it 100? Is it 200 hours?"
4. **Propose credit allocation** - Work with CSAM to earmark for GitHub
